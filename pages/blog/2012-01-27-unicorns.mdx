At [ModCloth](http://www.modcloth.com/), our website uses the [Ruby on Rails](http://rubyonrails.org/) framework, and we use the [unicorn web server](http://unicorn.bogomips.org/). When deploying code via [Capistrano](http://capify.org), we have a funny line of code which turns off our old web-servers and starts up new ones. Every time we deploy, we must kill a Unicorn.

This inspired a bit of artwork.
